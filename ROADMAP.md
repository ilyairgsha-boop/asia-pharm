## 🎯 Критические точки проверки

### Checkpoint 1: После установки инструментов
```bash
node --version    # Должно быть v18+
git --version     # Любая версия
supabase --version # Любая версия
```

### Checkpoint 2: После связывания с Supabase
```bash
ls -la .supabase  # Должна существовать
cat .supabase/config.toml | grep project_id  # Должен быть hohhzspiylssmgdivajk
```

### Checkpoint 3: После создания таблиц
```sql
-- В Supabase SQL Editor:
SELECT table_name FROM information_schema.tables 
WHERE table_schema = 'public';
-- Должны быть: profiles, products, kv_store_a75b5353
```

### Checkpoint 4: После деплоя Edge Function
```bash
curl https://hohhzspiylssmgdivajk.supabase.co/functions/v1/make-server-a75b5353/
# Ожидаемый ответ: {"status":"OK","message":"Asia-Pharm Store API"}
```

### Checkpoint 5: После деплоя на Vercel
```
Откройте Production URL в браузере
Должна загрузиться главная страница
Не должно быть ошибок в консоли браузера
```

### Checkpoint 6: После создания админа
```sql
SELECT email, is_admin FROM profiles WHERE is_admin = true;
-- Должен быть ваш email с is_admin = true
```
